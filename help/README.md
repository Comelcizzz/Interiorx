# INTERIORIX — опис для дипломної записки

Ця папка зібрана для **тексту дипломної записки**: що за система, як вона працює, яка бізнес-логіка і який технологічний стек. Це **не** інструкція до захисту й не шаблон розділів диплома.

Технічні деталі для розробників — у папці [`docs/`](../docs/).

---

## Назва

**INTERIORIX** — веб-система керування проєктами **інтер’єру та екстер’єру**: від першого звернення клієнта до кошторису, оплати, чека та відгуку.

---

## Суть проєкту

Студія дизайну отримує заявки з різних каналів, веде об’єкти по етапах, готує кошторис, приймає оплату й показує клієнту прозорий прогрес у **особистому кабінеті**. Замість розрізнених таблиць, листів і чатів — **одна платформа** з трьома зонами: публічний сайт (маркетинг), клієнтський портал і внутрішній робочий простір команди.
